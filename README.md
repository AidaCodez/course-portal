## 📖 Overview

This project is a simple Learning Management System (LMS) built to demonstrate how Next.js, React, Tailwind CSS, and GraphQL work together to create a modern full-stack web application. Users can browse available courses, subscribe for membership access, or register as instructors to manage course content.(In progress)

## 🚀 Features

- **Course Listing** – Display free and paid courses
- **Membership Access** – Subscribe as a member (TODO)
- **Instructor Access** – Register as an instructor (TODO)


## 🛠️ Tech Stack

- **Frontend:** React, JavaScript, HTML, CSS, Tailwind CSS
- **Backend:** Next.js, GraphQL
- **Other Tools:** Hygraph, shadcn/ui, Flaticon

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/AidaCodez/course-portal.git
cd course-portal
```

## ▶️ Running the Project

```
npm run dev
```
Open [http://localhost:3000/courses](http://localhost:3000) to view it in your browser.
